## Hostile / inert process-config fields supplied at create

GhosttyManualIsolationProbe deliberately fills stock surface-config process
fields with values that would be highly visible if honored:

  working_directory = /hive-manual-cwd-must-be-inert-and-missing
  command           = /bin/sleep 600
  env               = HIVE_MANUAL_ENV_MUST_BE_INERT=sentinel
  initial_input     = INITIAL_INPUT_MUST_BE_INERT
  wait_after_command = true

If command were launched, /bin/sleep 600 would remain as a direct child for the
entire create/use window (no race-to-exit). If initial_input were applied, the
sentinel string would appear on the screen before any process_output.
